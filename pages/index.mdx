# React gotchas

In this presentation, we will tackle few common problems that we can stumble upon when working in a React ecosystem
and some of the solutions that are available to us. List of the things that we will visit during this presentation:

- `setTimeout` and `setInterval`
- State updates after async functions
- Multiple request at same API endpoint
- Implement `AbortController`
- Globally handling errors
- Using `debounce`
