# More content

Writing custom hooks while making sure we cover all the edge cases is not an easy and straightforward task. This
presentation only covered manually written custom hooks, but there is a huge community behind writing custom hooks and
making them shareable:

[awesome-react-hooks](https://github.com/rehooks/awesome-react-hooks)

I would always recommend using eslint hooks plugin and make your goal to tackle even the slightest warning that this
plugin throws. No matter how small and insignificant warning is, most of the times there is a better solution out there
to solve the given problem:

[eslint-plugin-react-hooks](https://www.npmjs.com/package/eslint-plugin-react-hooks)
